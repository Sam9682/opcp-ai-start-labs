# Billing and Cost Tracking

## Objective

Learn how to monitor, analyze, and manage costs on the AI-Powered-Store platform by viewing current resource consumption, applying cost calculation rules to compute expected charges, configuring budget alerts for proactive cost management, and generating comprehensive usage reports.

## Prerequisites

- Installation on Bare-Metal Ubuntu (install-bare-metal)
- Starting Applications (starting-applications)

## Exercises

| # | Exercise | Objective |
|---|----------|-----------|
| 1 | View Resource Consumption | Query and display current resource consumption data for active containers |
| 2 | Calculate Expected Costs | Apply cost calculation rules to a given resource set and verify charges within 1% tolerance |
| 3 | Set Budget Alerts | Configure budget alert thresholds and verify alert triggering within 30 seconds |
| 4 | Generate Usage Reports | Generate a comprehensive usage report with resource name, quantity, unit cost, and total cost |
