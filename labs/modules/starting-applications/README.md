# Starting Applications

## Objective

Learn how to start deployed applications on the AI-Powered-Store platform using the CLI, REST API, and verify that applications are running correctly with health checks.

## Prerequisites

- [Adding New Applications](../adding-applications/README.md) — You must have successfully registered at least one application before starting it.

## Exercises

| # | Exercise Name | Objective |
|---|--------------|-----------|
| 1 | Start Application via CLI | Launch a registered application using the `aipoweredstore_cli.py` command-line tool |
| 2 | Start Application via REST API | Start an application by sending a POST request to `/api/deployments` with action "start" |
| 3 | Verify Running Status and Health Check | Confirm the application reaches a running state and responds to health checks on exposed ports |
