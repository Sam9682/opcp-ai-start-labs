Modification d'applications avec l'IA

Intermédiaire ⏱ ~90 min

Apprenez à utiliser les outils assistés par IA pour modifier, personnaliser et étendre les applications déployées sur la plateforme.

Prérequis

  • Terminé : Démarrage des applications
  • Amazon Kiro CLI installé et configuré
  • Une application en cours d'exécution à modifier

Ce que vous apprendrez

  • Utilisation du Kiro CLI pour les modifications assistées par IA
  • Application de changements aux applications en cours d'exécution
  • Test des modifications dans des environnements isolés
  • Annulation des changements si nécessaire
  • Bonnes pratiques pour le développement assisté par IA

Workflow assisté par IA

Le workflow typique pour modifier des applications avec l'IA comprend :

  1. Identifier l'application et les changements nécessaires
  2. Utiliser le Kiro CLI pour décrire vos modifications
  3. Réviser les changements de code générés
  4. Appliquer et tester dans un environnement de staging
  5. Déployer en production une fois validé
Labo disponible : Un module de laboratoire pratique accompagne cette leçon. Lancez le labo pour vous exercer dans un environnement isolé.
