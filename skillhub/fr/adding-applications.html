Ajout de nouvelles applications

Débutant ⏱ ~60 min

Apprenez à enregistrer et déployer des applications sur la plateforme AI-Powered-Store en utilisant le CLI, l'API REST et l'interface web.

Prérequis

  • Terminé : Installation sur Ubuntu Bare-Metal
  • Instance AI-Powered-Store en fonctionnement

Ce que vous apprendrez

  • Enregistrement d'applications via le CLI (aipoweredstore_cli.py)
  • Enregistrement d'applications via l'API REST
  • Enregistrement d'applications via l'interface web
  • Configuration des métadonnées d'application
Labo disponible : Un module de laboratoire pratique accompagne cette leçon. Lancez le labo pour vous exercer dans un environnement isolé.
