Installation on Bare-Metal Ubuntu

Beginner ⏱ ~120 min

Learn how to deploy the AI-Powered-Store platform on a fresh bare-metal Ubuntu server (22.04 or 24.04 LTS).

Prerequisites

  • A clean Ubuntu 22.04 or 24.04 LTS installation
  • Root or sudo access
  • At least 4 GB RAM and 20 GB disk space
  • Internet connectivity for package downloads

What You Will Learn

  • Verifying system prerequisites
  • Installing Docker Engine and docker-compose v2
  • Cloning the platform repository
  • Configuring the environment (deploy.ini and env_setup.sh)
  • Starting the platform with deployControlPlan.sh

Exercises

This lesson includes hands-on lab exercises. Complete each step and validate your progress.

Lab Available: A hands-on lab module accompanies this lesson. Start the lab to practice in an isolated environment.
