Billing & Cost Tracking

Advanced ⏱ ~60 min

Learn how to monitor resource consumption, track costs per application, and set up billing alerts for the platform.

Prerequisites

  • Completed: Installation on Bare-Metal Ubuntu
  • Running platform with multiple applications
  • Admin access to the platform dashboard

What You Will Learn

  • Understanding the cost model (CPU, RAM, GPU, storage)
  • Viewing per-application resource usage reports
  • Configuring billing alerts and thresholds
  • Exporting usage data for external reporting
  • Optimizing costs with serverless and auto-scaling

Cost Model Overview

The platform tracks resource usage across these dimensions:

  • CPU — Measured in core-hours
  • RAM — Measured in GB-hours
  • GPU — Measured in GPU-hours (per MIG slice)
  • Storage — Measured in GB-months (local + S3)
  • Network — Measured in GB transferred

Viewing Usage Reports

Access the admin dashboard to view per-application cost breakdowns and set up automatic reporting schedules.
